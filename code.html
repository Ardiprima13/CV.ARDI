Code Explanation

Learn the basics of web development

HTML (HyperText Markup Language)

HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web. Dengan HTML, kita bisa membuat struktur dasar sebuah halaman web seperti teks, gambar, dan elemen-elemen lain seperti form dan tabel.

  • Elemen Dasar HTML: <html>, <head>, <body>, <h1>, <p>, <a>.
  • Fungsi: Menentukan struktur halaman web.

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan visual halaman web. Dengan CSS, Anda dapat mengubah warna, font, margin, dan penataan elemen HTML lainnya di halaman web.

  • Selektor CSS: ID, class, elemen.
  • Properti CSS: `color`, `font-size`, `padding`, `border`.

JavaScript (JS)

JavaScript adalah bahasa pemrograman yang memungkinkan Anda untuk menambahkan interaktivitas di halaman web. JS dapat digunakan untuk memanipulasi DOM, mengatur event, dan membuat aplikasi web dinamis.

  • Fungsi JavaScript: Manipulasi elemen HTML, validasi form, interaksi dengan server (AJAX).
  • Contoh sintaks: `console.log()`, `alert()`, `document.querySelector()`.

PHP (Hypertext Preprocessor)

PHP adalah bahasa pemrograman server-side yang digunakan untuk membuat halaman web dinamis. PHP sering digunakan untuk mengelola database, memproses form, dan melakukan operasi server-side lainnya.

  • Fungsi PHP: Memproses form, berinteraksi dengan database, mengelola sesi pengguna.
  • Contoh sintaks: `echo`, `$_POST`, `mysqli_connect()`.
